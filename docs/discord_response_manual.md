# Discordで返答を読む方法

このボットは、
Twitch には短い案内だけ出して、
詳しい返答は Discord に出すことがあります。

見る流れはとても簡単です。

## まず覚えること

1. Twitch でコマンドを打つ
2. Twitch に短い案内が出る
3. Discord の指定チャンネルで詳しい返答を見る

Twitch に
`詳細はDiscord`
と出たら、
詳しい内容は Discord 側にあります。

## いちばん簡単な見方

### スマホの場合

1. Twitch でコマンドを打つ
   例: `!状態` `!探索 結果` `!探索 戦闘`
2. Twitch の返答を確認する
3. Discord アプリを開く
4. 配信者の Discord サーバーを開く
5. 案内されているチャンネルを開く
6. いちばん新しい投稿を見る

### PCの場合

1. Twitch でコマンドを打つ
2. Twitch の返答を確認する
3. Discord を開く
4. 配信者の Discord サーバーを開く
5. 指定チャンネルを開く
6. いちばん下の新しい投稿を見る

## どこを見ればいいの？

詳しい返答は、
配信者が設定した Discord のチャンネルに出ます。

よくある見つけ方はこの3つです。

1. チャンネル名に `bot` `rpg` `詳細` `log` が入っている場所を探す
2. サーバーのお知らせや固定メッセージを見る
3. 配信概要欄やパネルに書かれた案内を見る

## Twitchには何が出るの？

Twitch には、
短い案内だけが出ます。

例:

- `Alice 探索結果 / 通常 朝の森 / 帰還 / 3戦 / 詳細はDiscord`
- `Alice 探索戦闘 / 全3戦 / 詳細はDiscord`
- `Alice 攻略 / 次は朝の森 / 詳細はDiscord`

このあと Discord を開くと、
詳しい結果、
戦闘ログ、
おすすめ行動などを読めます。

## Discordでは何が見られるの？

コマンドによって変わりますが、
たとえば次のような内容が出ます。

- `!状態`
  レベル、HP、装備、素材、次のおすすめ
- `!探索 結果`
  経験値、ゴールド、ドロップ、帰還理由、探索要約
- `!探索 戦闘`
  何戦目で何が起きたか、ターンごとのログ
- `!攻略`
  今やると良いことと、その理由
- `!探索 履歴`
  直近の探索の流れ

## 迷った時はこれ

### Twitch に返答は出たのに Discord に見当たらない

- 少し待ってから更新する
- いちばん新しい投稿までスクロールする
- 別の bot 用チャンネルがないか探す

### Discordサーバーに入っていない

- Twitch で `!discord` を打って参加URLを確認する
- 配信者の招待リンクから参加する
- 参加後にチャンネル一覧を開く

### どのチャンネルかわからない

- 配信者に「詳細ログはどの Discord チャンネルですか？」と聞く

## 視聴者向けの最短手順

1. `!状態` を打つ
2. Twitch に `詳細はDiscord` と出たら Discord を開く
3. bot 用チャンネルの最新投稿を見る

これだけで読めます。
