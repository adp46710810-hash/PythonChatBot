# 配信概要欄用ヘルプ

スマホでも読みやすいように、短い行で切ったコピペ用ヘルプです。

## 短め版

RPG参加コマンド

`!ヘルプ`
使い方を見る

`!攻略`
今のおすすめ行動を見る

`!攻略順`
おすすめ攻略順を見る

`!me`
今の状態を見る

`!探索地`
行ける場所と役割を見る

`!探索診断 慎重 朝の森`
危険度と期待報酬を見る

`!探索開始 朝の森`
通常探索で出発

`!探索開始 慎重 朝の森`
安全寄りで出発

`!自動周回開始 通常 朝の森`
自動周回を開始

`!探索結果`
探索結果を受け取る

`!前回結果`
受取済みの前回結果を見る

`!履歴`
直近の探索履歴を見る

`!戦利品`
直近の主なドロップを見る

`!戦闘詳細`
戦闘の詳細を見る

`!戦闘詳細 3`
3戦目だけ見る

`!装備`
今の装備を見る

`!整理`
装備更新と不要装備整理をまとめて実行

`!保護 1`
残したい装備を保護

`!ポーション購入`
ポーション補充

`!強化 武器`
装備を強化

`!エンチャント 防具`
装備に特殊効果

`!蘇生`
戦闘不能から復帰

## 概要欄コピペ版

RPG参加方法
`!ヘルプ` で全体の使い方
`!me` で今の状態確認
`!探索地` で行き先確認
初回報酬つきの行き先も分かる
`!探索開始 朝の森` で出発
探索後は `!探索結果`
詳しく見るなら `!戦闘詳細`
進行目安は `!攻略`
迷ったら `!探索診断`

よく使うコマンド
`!装備`
`!整理`
`!バッグ`
`!素材`
`!前回結果`
`!履歴`
`!戦利品`
`!ポーション購入`
`!強化 武器`
`!エンチャント 武器`
`!自動周回開始 通常 朝の森`
`!蘇生`
`!探索診断 慎重 朝の森`

探索モード
慎重: 安全寄り
通常: 標準
強行: 高リスク高報酬

例
`!探索開始 朝の森`
`!探索開始 慎重 朝の森`
`!戦闘詳細 3`

## 少し詳しい版

はじめての人向け

1. `!me`
2. `!探索地`
   行き先と初回報酬を確認
3. `!探索開始 朝の森`
4. 終わったら `!探索結果`
5. `!整理`
6. さらに見るなら `!戦闘詳細`
7. 迷ったら `!攻略`
8. 行く前に `!探索診断 慎重 朝の森`

装備まわり

`!装備`
装備中の内容を見る

`!整理`
持ち物の整理と装備更新をまとめて実行

`!バッグ`
拾った装備を見る

`!保護 1`
残したい装備を保護

`!素材`
強化素材を見る

育成まわり

`!ポーション購入`
ポーションを買う

`!強化 武器`
武器を強化

`!強化 防具`
防具を強化

`!強化 装飾`
装飾を強化

`!エンチャント 武器`
武器に特殊効果

`!エンチャント 防具`
防具に特殊効果

`!エンチャント 装飾`
装飾に特殊効果

自動周回

`星影の祭壇` の導線解放と
欠片3個で自動周回解放

開始は
`!自動周回開始 通常 朝の森`

旧入力の
`!探索開始 自動 朝の森`
も使えます

止める時は
`!自動周回停止`

## 攻略順の目安

おすすめの進め方

1. `朝の森`
序盤の装備集め
Lv5までは不足装備を揃えやすい

2. `三日月廃墟`
`三日月呪石` 集め
エンチャントの準備

3. `ヘッセ深部`
高レア装備を狙う
拾ったら `!整理`

4. `紅蓮の鉱山`
`武器黒石` 集め
火力不足ならここ優先

5. `沈黙の城塞跡`
`防具黒石` 集め
倒されるならここ優先

6. `星影の祭壇`
`装飾黒石` 集め
装飾強化で周回効率アップ

周回のコツ

探索ごとに
`!探索結果`
`!整理`

勝てない時は
`慎重`

安定したら
`通常`

背伸びしたい時だけ
`強行`
