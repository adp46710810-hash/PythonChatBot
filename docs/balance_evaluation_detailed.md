### 【優先度2】強化・エンチャントコスト曲線の緩和 （実装難度：高）

**現在の問題**: コストが Lv に対して指数関数的に増加

**改善案**:

```python
# 現在（rules.py）
ENHANCEMENT_GOLD_COST_BASE = 50
ENHANCEMENT_MATERIAL_COST_BASE = 10
ENHANCEMENT_GOLD_COST_STEP = 3

# Lv20時の総コスト例:
# gold = 50 + (20 × 3) = 110
# material = 10 + (20 × 1) = 30

# 提案：段階的な緩和
def calculate_enhancement_cost(current_level, slot_type):
    base_gold = 50
    base_material = 10
    
    if current_level <= 20:
        # 序中盤：現在通り
        gold_step = 3
        material_step = 1
    elif current_level <= 35:
        # 後盤：段階的に緩和
        gold_step = 2.5  # ↓
        material_step = 0.8  # ↓
    else:
        # 終盤：さらに緩和
        gold_step = 2.0  # ↓↓
        material_step = 0.6  # ↓↓
    
    total_gold = base_gold + (current_level * gold_step)
    total_material = base_material + (current_level * material_step)
    
    return total_gold, total_material
```

**効果**: 後盤の強化進行が加速 ✅
**懸念**: 全体的なパワーインフレ（要調整）
